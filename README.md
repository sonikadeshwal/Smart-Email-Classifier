An NLP-powered email classifier that automatically sorts emails into **Spam**, **Promotion**, **Important**, and **Urgent** using TF-IDF vectorization and a Naive Bayes classifier.

---

## 🚀 Quick Start

```bash
git clone https://github.com/your-username/smart-email-classifier
cd smart-email-classifier
pip install -r requirements.txt
python setup_model.py        # train & save the model
streamlit run app.py
```

---

## 🧠 How It Works

1. **Preprocessing** — Lowercase, remove URLs/emails/punctuation
2. **TF-IDF Vectorization** — Unigrams + bigrams, 8000 features
3. **Naive Bayes** — Multinomial NB with 5-fold cross-validation
4. **Real-time prediction** — Returns label + probability breakdown

## 📊 Performance

| Metric | Score |
|---|---|
| CV Accuracy | 89% ± 5% |
| Categories | Spam, Promotion, Important, Urgent |
| Validation | 5-fold stratified cross-validation |
